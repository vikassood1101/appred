APPred Webserver


This webserver has been designed to assist scientists in the designing of antiprotozoal peptides.
This page helps the user on how different modules of this webserver works. This server's key modules are as follows:


Predict

This module allows the users to predict antiprotozoal peptides from a set of peptides or a peptide library (virtual screening). The server will predict and rank peptides based on their potential for antiprotozoal activity.



Design

This module generates mutants of a given peptide and predicts antiprotozoal peptides among the mutants. This server helps users to find the bare minimum number of mutations needed to boost the potency of query peptide.



Protein Scan

This module generates overlapping peptides from the query protein sequence and estimates their antiprotozoal activity. It helps the users to determine which segments of a protein have antiprotozoal activity.
